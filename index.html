One-time setup steps
  1. Go to console.cloud.google.com → create a project (e.g. “FiB Tracker”).
  2. APIs & Services → Enable APIs → enable Google Calendar API.
  3. OAuth consent screen → External → fill app name + your e-mail → add yourself as a Test user.
  4. Credentials → Create credentials → OAuth client ID → type Web application.
  5. Under “Authorized JavaScript origins” add: https://murtazachandio14.github.io
  6. Copy the Client ID → paste it in the box above → Save → Connect.
